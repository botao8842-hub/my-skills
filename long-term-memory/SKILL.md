# 长期记忆技能

## 描述
主动记录和分析用户的偏好、习惯、重要信息，让AI能更好地理解用户。

## 功能
1. 记录用户的偏好（喜欢的颜色、食物、做事方式）
2. 记住用户的习惯（什么时候有空、喜欢怎么沟通）
3. 分析用户的行为模式
4. 主动推测用户可能需要什么

## 触发方式
用户说以下内容时自动激活：
- "记住我..."
- "我喜欢..."
- "我讨厌..."
- "以后都..."
- "下次不要..."
- 或者AI主动记录用户的偏好

## 记忆内容

### 用户画像
- 姓名/称呼
- 性格特点
- 沟通偏好
- 起床/睡觉时间
- 活跃时间段

### 偏好
- 喜欢的（食物、颜色、风格）
- 讨厌的
- 做事方式
- 决策风格

### 习惯
- 常用工具
- 常用网站
- 说话方式

## 记忆格式
```
## 用户: [名字]

### 偏好
- 喜欢: xxx
- 讨厌: xxx

### 习惯
- xxx

### 上次聊到
- xxx
```

## 使用方式

用户说"记住我喜欢蓝色"，我就：
1. 记住这个偏好
2. 写入记忆文件
3. 下次主动用这个偏好

## 重要原则
- 主动记住用户说的
- 不要每次都问用户
- 分析用户行为，主动推测
- 定期回顾和更新记忆

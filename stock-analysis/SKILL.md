# SKILL.md - 股票分析助手

## 描述
分析A股股票，提供实时行情、技术分析和投资建议。用户说出股票代码或名称即可分析。

## 触发方式
用户说以下内容时：
- "分析[股票]"
- "[股票]怎么样"
- "看看[股票代码]"
- "推荐股票"

## 功能
1. **实时行情** - 通过网页抓取获取实时价格
2. **技术分析** - 判断短期趋势（基于涨跌幅）
3. **投资建议** - 给出买入/卖出/观望建议
4. **风险提示** - 设定止损位

## 数据来源
从财经网站抓取公开数据

## 使用方式
直接说"分析[股票]"，例如：
- 分析宁德时代
- 看看300750
- 茅台能买吗

## 输出格式
```
📊 [股票名称] ([代码])

【实时行情】
当前价：xxx
涨跌幅：+x.xx%
涨跌额：+x.xx
最高/最低：xxx/xxx
成交量/成交额

【技术分析】
短期趋势：🔥上涨 / 📉下跌 / ➡️震荡
支撑位：xxx
压力位：xxx

【投资建议】
建议：买入/卖出/观望
止损位：xxx（-5%）
止盈位：xxx（+10%）

⚠️ 免责声明：仅供参考，不构成投资建议，投资有风险，入市需谨慎
```

## 注意事项
- 投资有风险，入市需谨慎
- 仅供参考，不构成买卖建议
- 建议配合其他信息综合判断
- 务必设定止损位保护本金

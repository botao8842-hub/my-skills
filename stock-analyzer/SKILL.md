# SKILL.md - 股票分析

## 描述
分析A股/港股个股实时行情，判断大概率上涨还是下跌，给出参考建议。

## 触发方式
用户说：
- "分析[股票代码]"
- "看看[股票]怎么样"
- "XXX股票能买吗"
- "查询[代码]行情"

## 功能
1. 查询个股实时行情（价格、涨跌幅、成交量）
2. 判断短期走势
3. 给出参考建议（买/卖/观望）
4. 设定止损位

## 使用方式
直接说"分析[股票代码]"

## 输出格式
```
📊 [股票名称]

【实时行情】
当前价：xxx
涨跌幅：+x.xx%
最高/最低：xxx/xxx
成交量/成交额

【技术分析】
短期趋势：上涨/下跌/震荡
支撑位/压力位

【参考建议】
建议：买入/卖出/观望
止损位：xxx

⚠️ 免责声明
```

## 注意事项
- 仅参考，投资有风险
- 设定止损保护本金

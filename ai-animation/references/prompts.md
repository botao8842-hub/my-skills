# AI动画提示词模板

## 一、基础结构

```
[主体] + [动作] + [场景] + [风格] + [运镜] + [光线]
```

## 三、模板库

### 模板1: 人物特写

```
[人物], [表情], [动作], close-up shot, [光线], [风格], 4k, high quality
```

**例子**:
```
A beautiful woman smiling, looking at camera, close-up shot, cinematic lighting, soft focus, 4k
```

### 模板2: 场景漫游

```
[场景], [物体], [运动], aerial view, [光线], [风格], 4k
```

**例子**:
```
A futuristic city with neon lights, flying cars, aerial view, cyberpunk style, cinematic lighting, 4k
```

### 模板3: 动物/生物

```
[动物], [动作], [场景], [风格], [光线], 4k
```

**例子**:
```
A majestic dragon flying over mountains, fire breath, fantasy style, epic lighting, 4k
```

### 模板4: 数字人

```
A professional [性别] anchor, wearing [服装], [背景], news studio lighting, high definition, realistic
```

**例子**:
```
A professional female news anchor wearing business suit, sitting in modern office, studio lighting, high definition, realistic
```

## 四、常用词组

### 提升质量

- `high quality` - 高质量
- `detailed` - 细节丰富
- `8k` - 超高清
- `masterpiece` - 杰作
- `photorealistic` - 照片级写实
- `cinematic` - 电影感

### 增加动态

- `dynamic` - 动态感
- `action frozen` - 动作冻结
- `motion blur` - 动态模糊
- `explosion` - 爆炸效果

### 艺术风格

- `wall-e style` - 瓦力风格
- `disney style` - 迪士尼风格
- `ghibli style` - 吉卜力风格
- `tear of steel` - 铁甲钢拳风格

---

## 五、注意事项

1. **主体要明确**: "a robot" 比 "something" 好
2. **动作要具体**: "running fast" 比 "moving" 好
3. **风格要统一**: 不要混用冲突的风格
4. **用逗号分隔**: 关键词之间用逗号
5. **语言**: 英文效果通常更好

---

## 六、中文提示词技巧

如果用中文：

1. **简短直接**: "一只猫在草地上跑"
2. **加风格词**: "皮克斯风格"、"赛博朋克"
3. **加质量词**: "高清"、"电影感"

可灵对中文支持较好，其他工具建议用英文。
